## Overview

Hadie Danker provides the official connector to the WhatsApp Business API. You integrate it into your applications to send and receive messages at scale. Businesses use it for customer support, notifications, and marketing campaigns through WhatsApp's secure platform.

This connector handles authentication, message templates, webhooks, and media attachments. You get reliable delivery reports and conversation analytics without managing infrastructure.

## Quick Start

Get started in minutes.

<Steps>
  <Step title="Install SDK" icon="download">
    <CodeGroup tabs="Node.js,Python">
```javascript
npm install hadie-danker
```
````python
pip install hadie-danker
````
    </CodeGroup>
  </Step>
  <Step title="Set API Key" icon="key">
    Get your key from `https://dashboard.example.com/api-keys`.
  </Step>
  <Step title="Send First Message" icon="send">
    ```javascript
import { HadieDanker } from 'hadie-danker';

const client = new HadieDanker({ apiKey: `YOUR_API_KEY` });
await client.messages.send({
  to: `whatsapp:+1234567890`,
  type: `text`,
  text: { body: `Hello from Hadie Danker!` }
});
    ```
  </Step>
</Steps>
