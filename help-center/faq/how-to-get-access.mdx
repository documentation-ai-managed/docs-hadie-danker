title: How do I get access to the WhatsApp Business API?
description: How to request and activate API access with Hadie Danker.
---

## Overview

You need an approved WhatsApp Business Profile.

## Steps

1. Register at `https://dashboard.hadiedanker.com`
2. Complete business verification with Facebook
3. Request API access from the dashboard
4. Await activation email

## After Approval

- Get your API key in "Settings"
- Use it for all API requests

<Callout kind="info">Contact support if approval takes more than 3 days.</Callout>
