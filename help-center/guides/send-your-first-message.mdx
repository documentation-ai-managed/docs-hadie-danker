---
title: Send Your First WhatsApp Message
description: Simple guide to sending a WhatsApp message using Hadie Danker API.
---

## Steps

<Steps>
  <Step title="Get your API key">
    Log in at `https://dashboard.hadiedanker.com`, then copy your API key from Settings.
  </Step>
  <Step title="Send a test message">
    ````bash
    curl -X POST https://api.hadiedanker.com/v1/messages \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{"to": "+1234567890", "text": "Hello from Hadie Danker!"}'
    ````
  </Step>
  <Step title="Check delivery">
    Use your dashboard to view message status and logs.
  </Step>
</Steps>
